# AstrBot 群u语录插件
什么？群友又爆典了！当然是截图保存让他名垂千古口牙（

## 使用方法
发送“语录投稿+图片”，bot会将群u语录保存到本地（不同群聊相互隔离）  
如果你的qq未开启半屏相册发图功能，可以通过【qq左划→设置→通用→图片、视频→发图方式】进行修改  
权限系统可以通过发送“投稿权限+模式数字”进行设置  
   0：关闭投稿系统  
   1：仅管理员可投稿  
   2：全体成员均可投稿  
   
在bot被戳一戳时，将会随机选择一张黑历史发送（有10秒默认冷却）  
如果想要修改这个冷却，请发送“戳戳冷却+冷却时间”来设置，单位为秒  

## 版本更新v1.4
管理员权限获取方式优化：
移除了硬编码读取 data/cmd_config.json 的方式。
改为使用 AstrBot 标准接口 context.get_config().get("admins_id") 直接从配置文件加载管理员列表。
投稿功能增强 (支持引用)：
新增了 引用(回复)投稿 功能：现在可以回复别人的图片发送“语录投稿”进行保存。
解析逻辑修复：针对不同协议端（OneBot V11）返回的引用消息格式（JSON数组或CQ码字符串）进行了兼容处理，增加了正则匹配逻辑，解决了无法识别引用图片的问题。
图片下载逻辑优化：
重构了 download_image 方法。
增加容错机制：如果无法从本地缓存读取（例如引用消息的情况），会自动尝试通过 API 获取路径，最后尝试通过 URL 下载。
新增主动触发指令：
新增了 /语录 或 语录 指令，允许群友不通过“戳一戳”也能主动请求一张随机语录。

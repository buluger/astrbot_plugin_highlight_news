# AstrBot 群友精华插件

记录群里的「精华」消息（**文字或图片**），之后可以随机回味，或把全部条目汇总成 **一张长图** 查看。不同群聊数据相互隔离。

注：由于我未接触过Python语言，本项目所有代码均由cursor生成，暂时无法保证功能完整性，还在测试阶段

## 依赖

- 随机回顾、戳一戳、图片投稿：无额外要求。
- **「精华图」汇总长图**：需要安装 Pillow，例如：
  ```bash
  pip install Pillow
  ```
  或在 AstrBot 环境中安装本目录下的 `requirements.txt`。

## 使用方法

### 投稿（精华）

在开启投稿权限的前提下，发送 **`精华投稿`**：

| 方式 | 说明 |
|------|------|
| **文字** | `精华投稿 后面写要保存的文字` |
| **图片** | 带图发送 `精华投稿`，或 **引用** 一条含图的消息后发送 `精华投稿` |
| **引用文字** | **引用** 别人的纯文字消息后发送 `精华投稿`（不写额外内容时，保存被引用的文字） |

数据保存在 **`data/highlights_data/<群号>/`**，索引文件为 **`highlights.json`**；图片文件保存在同目录下。

### 权限与戳戳冷却

通过 **`投稿权限+模式数字`** 设置（需 AstrBot 配置中的 **bot 管理员**）：

- `0`：关闭投稿系统  
- `1`：仅管理员可投稿  
- `2`：全体成员均可投稿  

**戳一戳** 时会随机出一条精华（文字则发文字，图片则发图）；默认有冷却防刷屏（约 10 秒）。  
修改冷却：发送 **`戳戳冷却+秒数`**。

若 QQ 发图异常，可在 **QQ → 设置 → 通用 → 图片、视频 → 发图方式** 中调整发图方式。

### 指令一览

| 指令 | 作用 |
|------|------|
| **`/精华`** 或 **`精华`** | 随机发送一条精华（文字或图片） |
| **`精华图`** / **`/精华图`** | 生成一张 **长图**，列出当前群内全部精华（文字折行展示，图片条目显示为 `[序号] [图片]`） |
| **`精华列表`** / **`/精华列表`** | 与「精华图」相同 |

## 版本

当前版本：**v1.0**。

## 注意事项

投稿条数与内容无内置审核上限，请群管理自行约定规则；汇总图仅作列表展示，不会在一张图里嵌入所有原图缩略图（避免体积与排版问题）。
